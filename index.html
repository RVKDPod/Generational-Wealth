Module 06 · BLS Regional Data
Regional Cost of Living
BLS tracks CPI separately for ~25 major metro areas using the CUURA/CUURS series codes. Multipliers show cost of living relative to the national average.
Data Source: Bureau of Labor Statistics Regional CPI series (CUURA___SA0). The BLS publishes metro-level CPI for ~25 city areas — some monthly, some bi-monthly. Regional multipliers are derived from BLS's own geographic cost-of-living factors. For programmatic access, use the free BLS Public Data API v2 with no API key required (series IDs shown below). For state-level affordability, housing data from HUD and Census is more granular than CPI.

How to Pull This Data Yourself — BLS API (No Key Required)

The BLS Public Data API v2 lets you fetch CPI series for any metro area. Series IDs follow the pattern CUURA[AREA]SA0 for unadjusted CPI or CUURS[AREA]SA0 for seasonally adjusted.

fetch('https://api.bls.gov/publicAPI/v2/timeseries/data/', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    seriesid: ['CUURA103SA0',   // Boston CPI
               'CUURA207SA0',   // Chicago CPI
               'CUURA421SA0',   // New York CPI
               'CUUR0000SA0'],  // National CPI
    startyear: '2010',
    endyear: '2024'
  })
})

Note on state-level data: The BLS does not publish state-level CPI (only metro areas). For state cost-of-living, use the Missouri Economic Research and Information Center (MERIC) composite index, or HUD's Fair Market Rents for housing-specific data by state and county.
